# 🚀 AI Resume Maker

An AI-powered Resume Maker that helps users create professional, ATS-friendly resumes with an intuitive interface and AI-assisted content generation. The application streamlines the resume creation process by allowing users to build, customize, preview, and export resumes efficiently.

## ✨ Features

- 🤖 AI-assisted resume content generation
- 📝 Create and edit professional resumes
- 🎨 Clean, responsive, and modern UI
- 📄 ATS-friendly resume formatting
- ⚡ Real-time resume preview
- 📥 Export resumes
- 🔒 Secure and user-friendly experience

## 🛠️ Tech Stack

**Frontend**
- React.js
- TypeScript
- Tailwind CSS
- Vite

**Backend**
- Supabase

**AI**
- Google Gemini API / LLM

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/sandeepgudasi/AI-Resume-maker.git
```

### 2. Navigate to the project

```bash
cd AI-Resume-maker
```

### 3. Install dependencies

```bash
npm install
```

### 4. Configure environment variables

Create a `.env` file in the project root.

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_key
VITE_GEMINI_API_KEY=your_gemini_api_key
```

### 5. Run the application

```bash
npm run dev
```

The application will be available at:

```
http://localhost:5173
```
